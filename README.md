# Drink Tracker (Water & Drinks Tracker)

Простий та зрозумілий Flutter-додаток для трекінгу води та напоїв з локальним збереженням даних і системою досягнень.

---

## Основні можливості

- Додавання склянок води(250 мл)
- Підрахунок денної гідратації
- Денна ціль (у мл)
- Історія по днях
- Система досягнень
- Локальне збереження 
- Працює на Web / Desktop / Mobile

---

## Архітектура проєкту


Проєкт побудований з чітким розділенням відповідальностей:
```
lib/
├─ main.dart // точка входу
├─ app.dart // тема та MaterialApp
├─ models/ // бізнес-моделі
├─ storage/ // локальне збереження
├─ state/ // глобальний стан
└─ ui/ // користувацький інтерфейс
```
